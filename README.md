# Responsive Layout Practice

This project is a responsive landing page built with **HTML and CSS**.  
It focuses on practicing **mobile-first design**, **Flexbox**, and **responsive layouts** using media queries.

## Features

- Mobile-first responsive design
- Navigation menu with burger menu on mobile
- Flexbox layout
- Tablet and desktop breakpoints
- Clean BEM-style class naming

## Technologies

- HTML5
- CSS3
- Flexbox
- Media Queries

## Purpose

This project was created as a **practice exercise to improve responsive layout skills** and build a clean, structured frontend layout.
